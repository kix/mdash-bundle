Mdash bundle
============
This is a Symfony 2 bundle for [E. Muravjov's typograph](http://mdash.ru).

Installation is simple. Just drop this line into your `composer.json`'s `require` section:
```
"kix/mdash-bundle": "0.5.x"
```
